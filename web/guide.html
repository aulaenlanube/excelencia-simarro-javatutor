Script: Dataset Preprocessor (Python)

cleaner.py 
import json

def format_to_alpaca(raw_data):
    # Transforma texto sucio del aula en pares de entrenamiento
    clean_corpus = []
    for item in raw_data:
        example = {
            "instruction": "Escribe un ejemplo de " + item['topic'],
            "input": "Contexto: DAM IES Simarro",
            "output": item['expert_explanation'] + "\n\n```java\n" + item['code'] + "\n```"
        }
        clean_corpus.append(example)
    
    with open('dataset_final.jsonl', 'w') as f:
        for line in clean_corpus:
            f.write(json.dumps(line) + '\n')

Consumo desde el Entorno Real

Uso desde Java (Client)

Los alumnos no usan la terminal para hablar con la IA. Hemos implementado un cliente Java estándar que consume la API generada por Ollama o vLLM. 

// Ejemplo de llamada desde una aplicación Java de escritorio
public class TutorClient {
    public static String getHelp(String error) {
        HttpClient client = HttpClient.newHttpClient();
        HttpRequest request = HttpRequest.newBuilder()
            .uri(URI.create("http://localhost:11434/api/generate"))
            .POST(BodyPublishers.ofString("{\"model\": \"javatutor-ia\", \"prompt\": \"" + error + "\"}"))
            .build();
        
        // Retorna la respuesta del Tutor optimizada para el alumno
        return client.send(request, BodyHandlers.ofString()).body();
    }
}
